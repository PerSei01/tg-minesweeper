# tg-minesweeper
Minesweeper Telegram bot with inline keyboard UI

## Current status
In development

## Tech stack
- Python 3.x
- aiogram

## Detailed description
This project is a learning and portfolio project.  
The goal is to implement a fully working Minesweeper game inside Telegram and gradually improve the architecture and functionality as the project grows.

The main focus is:
- clean game logic
- correct implementation of classic Minesweeper rules
- state management for multiple games
- incremental refactoring and feature growth

## Planned features (v0.1)
- 8x8 field with 10 mines  
- Safe first click  
- Inline keyboard UI  
- Win / Lose detection  
- Click / Flag mode  
- Time tracking

## Future plans
In future versions I plan to add:
- game progress saving  
- statistics and records  
- multiple difficulty levels  
- improved UI and controls
